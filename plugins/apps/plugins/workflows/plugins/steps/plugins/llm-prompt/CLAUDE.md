# llm-prompt

A workflow step type that uses an LLM as a step. It interpolates the configured
prompt against the previous step's output (`{{ path }}` reads a field, `{{ . }}`
is the whole input), runs a one-shot Claude generation via the local Claude CLI
(`runClaudePrint`) at the chosen model tier, and emits `{ text }` as the step
output.

Config: `{ tier?: string; system?: string; prompt?: string }` — `tier` defaults
to `"haiku"` and is validated against `MODEL_TIERS`; `system` is an optional
system prompt; `prompt` is the interpolated user prompt (empty → the executor
throws). `ClaudeCliError` is allowed to propagate (fail loudly).
