# jog-wheel

The shared chrome for Sonata's horizontal jog wheels — a bordered pill with a
leading icon, a draggable ribbed "physical wheel" face, and a right-hand
readout. It owns only the visuals so every jog wheel stays pixel-identical
(including the leading icon's clearance from the rounded corner); the value
_model_ stays with the consumer.

`JogWheel` (from the `web` barrel) takes the formatted `readout`, the `icon`,
ARIA slider metadata, a `ribOffsetPx` (bind to the value so the ribs slide as it
scrubs), and the `drag` handle returned by `useInertialDrag` (pointer handlers +
phase). The consumer decides the scrub space: `SpreadWheel` scrubs `log(spread)`
(a fixed pixel travel = a fixed zoom ratio); `TempoWheel` scrubs linear tempo
and reaches 0%. Pair it with the sibling `inertial-drag` primitive for the
grab → flick → coast → settle physics.
