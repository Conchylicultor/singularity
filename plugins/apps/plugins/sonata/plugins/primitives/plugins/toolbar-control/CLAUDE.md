# toolbar-control

The shared chrome for Sonata's toolbar "dial" controls — the speed/spread jog
wheels (`jog-wheel`) and the transpose stepper. A compact bordered pill with a
leading, muted category icon and a row of caller-supplied segments.

`ToolbarControl` (from the `web` barrel) owns the rounded border, the tooltip,
the no-song disabled dimming, and the leading icon's horizontal clearance from
the rounded corner — so its glyph never crowds the border (the bug this frame
exists to make structurally impossible across every toolbar dial). Pass the
already-sized `icon`, the `tooltip`, an optional `disabled`, and the segments as
`children`. Segment dividers (`border-l`/`border-x`) live on the segments, not
the frame, so each control divides its interior however it likes.
