# shell

App shell for the Mail app. Registers the `/mail` app entry (rail icon
`MdMail`, tooltip "Mail"), defines the `Mail.Sidebar` render slot for left-rail
mailbox/label entries, and registers the index pane (`mail-root`, empty segment
+ `appPath` so bare `/mail` lands here). The index pane is a capability-driven
empty-state: it reads `useGmailAccess()` from the Gmail integration and guides
the user to enable, connect, or grant access — the inbox list lands in a later
phase. Auth state comes only from `@plugins/integrations/plugins/gmail/web`,
never `@plugins/auth` directly.
