# text-editor

Generic Lexical-based rich text editor primitive — an extensible shell where
plugins inject features via the `TextEditor.Plugin` slot and
`registerNodeExtension`.

The editor manages two-way markdown-to-Lexical sync, submit behavior
(Enter / Cmd+Enter), and accepts an optional `bottomSlot` prop for UI
rendered inside the LexicalComposer boundary below the text area (so
slot components can call `useLexicalComposerContext()`).

Feature-specific behaviors live in sub-plugins that contribute via the
Plugin slot.

## Sub-plugins

- **`paste-images`** — Image paste/drop support. Registers an `ImageNode`
  extension and contributes the paste/drop Lexical command handler via
  `TextEditor.Plugin`. Also owns the attachment markdown helpers
  (`extractAttachmentIds`, `rewriteAttachmentMarkdown`, etc.).
