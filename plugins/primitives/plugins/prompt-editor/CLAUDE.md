# prompt-editor

Lexical-based prompt editor primitive — an extensible shell where plugins
inject features via the `PromptEditor.Plugin` slot and `registerNodeExtension`.

The editor manages two-way markdown ↔ Lexical sync, submit behavior
(Enter / Cmd+Enter), and a `FloatingAction` slot for overlay UI.
Feature-specific behaviors (image paste, templates, etc.) live in
sub-plugins that contribute via the Plugin slot.

## Sub-plugins

- **`paste-images`** — Image paste/drop support. Registers an `ImageNode`
  extension and contributes the paste/drop Lexical command handler via
  `PromptEditor.Plugin`. Also owns the attachment markdown helpers
  (`extractAttachmentIds`, `rewriteAttachmentMarkdown`, etc.).
