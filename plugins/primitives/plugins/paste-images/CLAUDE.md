# paste-images

Lexical-based prompt editor with paste-image support and rich thumbnails.
Pasted/dropped images are uploaded immediately via `uploadAttachment` and
rendered inline as `<AttachmentThumbnail>` elements (hover-× to remove,
click to expand into a lightbox).

Editor state serializes to markdown — images become
`![alt](/api/attachments/<id>)` references. Storage is plain markdown text;
ownership is tracked by the consumer via `Attachments.defineLink(ownerTable)`.
On agent-launch, callers replace the image markdown refs with `@<disk-path>`
via `rewriteAttachmentMarkdown` so Claude reads the images from disk.
