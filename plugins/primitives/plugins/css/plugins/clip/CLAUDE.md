# clip

The clipping box — hides overflow **without** scrolling. The `overflow-hidden`
sibling of `<Scroll>`, kept deliberately orthogonal: Scroll owns scrollable
overflow, Clip owns clipped overflow. ~52 call sites used raw `overflow-hidden`.

## Scope

Owns only the box-level clipping mechanic. Decoration stays in `className` —
`rounded-*` / `border` are not banned, so a rounded-clipped media box is `<Clip
className="rounded-md border">`.

**Not** for single-line text truncation (`overflow-hidden text-ellipsis
whitespace-nowrap`) → that is `<Text>` inside a line container, the dedicated
truncation leaf (it ellipsizes via the ambient single-line context).

## Props

- **`axis`** — `"both"` (default) | `"x"` | `"y"`.
- **`fill`** — emit the flex-child fill pair (`min-h-0 flex-1`, or `min-w-0 flex-1` on the `x` axis). Default false.
- **`as`** — host element. Default `div`. **`className`** composes last.
