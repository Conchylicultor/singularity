# line

The bare **line-container contract** — `flex region-line` (structural
`whitespace-nowrap`) paired with `SingleLineProvider value={true}` (the ambient
signal `<Text>` reads to ellipsize). No padding / height / hover / border of its
own: consumers layer chrome via `className`.

`Row` (interactive row chrome), `Bar` (toolbar strip chrome), and bespoke
single-line strips (the app tab chip, the transcript `CollapsibleCard` header)
COMPOSE `<Line>` instead of re-deriving the `region-line` + provider pair, so the
contract lives in one place. `Badge` stays on `inline-flex` (the inline-level
case) and intentionally does not compose it.
