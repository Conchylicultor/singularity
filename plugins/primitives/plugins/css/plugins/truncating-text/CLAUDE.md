# truncating-text

Single-line text that truncates instead of wrapping, plus the lint rule that
keeps single-line *chrome* (toolbars, headers, rows, chips) honest.

## Single-line is a property of the region, not the leaf

A horizontal chrome region — a toolbar, a card/pane header, a list row, a chip — is
single-line *by contract*. CSS wrapping is the default, so the durable fix is to set
`whitespace-nowrap` once on the **region primitive's root** (it inherits to every
descendant text node; a `truncate`/`TruncatingText` leaf still ellipsizes because it
re-declares nowrap). The shared single-line invariant lives in the `region-line`
`@utility` (`items-center` + `whitespace-nowrap`, in `ui-kit`'s `app.css`); the
region primitives bake it in: the `Bar` chrome strip (app/pane toolbars + pane
headers), `Row`, `CollapsibleTrigger`, and the `CollapsibleCard` header apply
`region-line` directly, and `PaneChrome`, `ToggleChip`, `SegmentedControl`,
`tabbed-view` tabs, `section-label`, `breadcrumb`, `Badge`, `Button` carry the same
guarantee. Don't re-solve wrapping per text leaf — put `whitespace-nowrap` (or
`region-line`) on the row.

## Enforcement: `no-clip-without-nowrap`

`lint/no-clip-without-nowrap.ts` fails `./singularity check` on the "overflow-hidden
trap": a horizontal flex row (`flex`/`inline-flex`) carrying `overflow-hidden` but no
`whitespace-nowrap`/`truncate`. That shape *looks* defended but isn't —
`overflow-hidden` clips text that has **already** wrapped to a second line; it does not
stop the wrap. Fix by adding `whitespace-nowrap` so children never wrap (let a
`truncate` leaf ellipsize).

The rule excludes genuine non-rows so nowrap is never forced where it would freeze
descendant wrapping: vertical stacks (`flex-col`), explicit multi-line (`flex-wrap`),
self-sizing-height layout regions (`h-full`/`h-screen`/`min-h-0`), and cross-axis
start/end/stretch alignment (`items-start`/`items-end`/`items-stretch` — a single-line
row centers its children). A genuine non-text clip (e.g. a morph-animation panel)
escapes per-site, travelling with the code:

```ts
// eslint-disable-next-line truncating-text/no-clip-without-nowrap -- reason
```
