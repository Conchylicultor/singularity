# grid

The responsive / uniform grid primitive. A wrapping grid of **equal-width
cells** that reflows by available width ‚Äî galleries, launcher grids, card
collections. Pick **either** the cell width (`minCellWidth`, responsive) **or** a
fixed column count (`cols`) ‚Äî the two are an **xor** discriminated union, never
both ‚Äî and the track function does the rest.

## Why this is a closed prop surface, not a `grid-template` passthrough

The structural `rigid | flexible | rigid` row ‚Äî the case that used to be
hand-rolled with per-call-site `flex-1` / `min-w-0` / `shrink-0` ‚Äî is **Frame's**
job. Grid owns the one remaining layout concern with no other home: a grid of
uniform cells. An arbitrary `grid-template` string *is* the raw CSS the layout
standard bans, so Grid deliberately exposes no `columns` escape hatch. The two
real needs are covered (structural ‚Üí `Frame`, uniform responsive ‚Üí `Grid`);
anything rarer stays a per-site `eslint-disable ‚Ä¶ -- <reason>` exception.

## The track function

`gridTemplateColumns(opts)` is the single source of truth ‚Äî exported so the
component and the pure test share one definition. Its parameter is the same
`{cols}` xor `{minCellWidth, mode}` union as the component, so the two
mutually-exclusive paths can't be conflated:

| Path                  | Output                                            | Behaviour                                                       |
| --------------------- | ------------------------------------------------- | -------------------------------------------------------------- |
| Fixed (`{ cols }`)    | `repeat(<cols>, minmax(0, 1fr))`                  | exactly N equal columns                                        |
| Responsive `fill`     | `repeat(auto-fill, minmax(<minCellWidth>, 1fr))`  | stable column count ‚Äî keeps empty trailing tracks              |
| Responsive `fit`      | `repeat(auto-fit, minmax(<minCellWidth>, 1fr))`   | collapses empty tracks so present cells stretch to fill the row|

The `minmax(0, ‚Ä¶)` / `minmax(<minCellWidth>, ‚Ä¶)` min lets a track shrink without
forcing the row to overflow.

## Props

The two geometry props are an xor discriminated union ‚Äî pass exactly one path:

- **Responsive path** ‚Äî **`minCellWidth`** (required, `"12rem"`-style min width
  before a row wraps) + optional **`mode`** (`"fill"` default, stable columns |
  `"fit"`, stretch present cells). No `cols` allowed.
- **Fixed path** ‚Äî **`cols`** (required, fixed column count). No
  `minCellWidth`/`mode` allowed.

Shared (both paths):

- **`gap`** ‚Äî spacing-ramp step. Default `md`.
- **`align`** / **`justify`** ‚Äî `StackAlign` / `StackJustify` (items / content).
- **`as`** ‚Äî host element. Default `div`. **`className`** composes last.
