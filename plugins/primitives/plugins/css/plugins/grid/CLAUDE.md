# grid

The responsive / uniform grid primitive. A wrapping grid of **equal-width
cells** that reflows by available width — galleries, launcher grids, card
collections. Pick the cell width (`minCellWidth`) or a fixed column count
(`cols`) and the track function does the rest.

## Why this is a closed prop surface, not a `grid-template` passthrough

The structural `rigid | flexible | rigid` row — the case that used to be
hand-rolled with per-call-site `flex-1` / `min-w-0` / `shrink-0` — is **Frame's**
job. Grid owns the one remaining layout concern with no other home: a grid of
uniform cells. An arbitrary `grid-template` string *is* the raw CSS the layout
standard bans, so Grid deliberately exposes no `columns` escape hatch. The two
real needs are covered (structural → `Frame`, uniform responsive → `Grid`);
anything rarer stays a per-site `eslint-disable … -- <reason>` exception.

## The track function

`gridTemplateColumns({ minCellWidth, mode, cols })` is the single source of truth
— exported so the component and the pure test share one definition. Two
mutually-exclusive paths:

| Path                  | Output                                            | Behaviour                                                       |
| --------------------- | ------------------------------------------------- | -------------------------------------------------------------- |
| Fixed (`cols != null`)| `repeat(<cols>, minmax(0, 1fr))`                  | exactly N equal columns; `cols` **wins** over `minCellWidth`   |
| Responsive `fill`     | `repeat(auto-fill, minmax(<minCellWidth>, 1fr))`  | stable column count — keeps empty trailing tracks              |
| Responsive `fit`      | `repeat(auto-fit, minmax(<minCellWidth>, 1fr))`   | collapses empty tracks so present cells stretch to fill the row|

The `minmax(0, …)` / `minmax(<minCellWidth>, …)` min lets a track shrink without
forcing the row to overflow.

## Props

- **`minCellWidth`** (required) — `"12rem"`-style min width before a row wraps.
- **`mode`** — `"fill"` (default, stable columns) | `"fit"` (stretch present cells).
- **`cols`** — fixed column count; when set it wins over the responsive path.
- **`gap`** — spacing-ramp step. Default `md`.
- **`align`** / **`justify`** — `StackAlign` / `StackJustify` (items / content).
- **`as`** — host element. Default `div`. **`className`** composes last.

## Tests
