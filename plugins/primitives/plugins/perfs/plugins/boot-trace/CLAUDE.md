# boot-trace

Dependency-light, web-only leaf library that owns the module-level boot-span
store. Imported eagerly by the framework boot path so it is live before any
resource mounts: framework instrumentation points open/close spans
(`startBootSpan` / `markBootInstant` / `recordBootSpan`) and a consumer reads the
assembled timeline via `getBootTrace()`, which also folds in Navigation/Paint
Timing, the first React commit, the main-thread **Long Tasks**, and the
**asset** Resource Timing (scripts/stylesheets). The Long Task + Resource Timing
data covers the pre-boot-trace window the span store can't reach — the bundle
download/parse/eval before this module even runs. Everything shares one clock
(`performance.now()` on `performance.timeOrigin`); `bootWindowEnd(trace)` is the
single source of truth for when boot ends (later of FCP / first commit), used to
clip long tasks to the boot window.
