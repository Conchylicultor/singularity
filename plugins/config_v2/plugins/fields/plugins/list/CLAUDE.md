# list

Sortable list field type for config_v2. Represents sorted collections of
structured sub-items with stable UUID identity and fractional-index ordering.

Each list item has an auto-injected `id` (UUID) and `rank` (fractional-index
string) plus the user-defined sub-fields. Consumers define lists via
`listField({ itemFields: { ... }, default: [...] })` — the `id` and `rank`
are transparent system fields injected at read time by the server.

The web renderer uses `SortableList` + `SortableItem` for drag-and-drop
reordering and dispatches to `FieldRenderer` for each item's sub-fields.
