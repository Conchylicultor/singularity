# enum

Enum field type for config_v2. Single-choice selection from a fixed set of
options. Renders as a radio group (≤3 options) or dropdown (≥4), with an
optional `display` override.

## Usage

```ts
import { enumField } from "@plugins/config_v2/plugins/fields/plugins/enum/core";

const myConfig = defineConfig({
  fields: {
    logLevel: enumField({
      label: "Log level",
      description: "Controls log verbosity",
      options: ["debug", "info", "warn", "error"],
      default: "info",
    }),
    theme: enumField({
      label: "Theme",
      options: [
        { value: "light", label: "Light" },
        { value: "dark", label: "Dark" },
      ],
      display: "radio",
    }),
  },
});
```
