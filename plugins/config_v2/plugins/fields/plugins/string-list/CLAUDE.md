# string-list

Plain string-array field type for config_v2. Stores a bare `string[]` and
passes through config_v2's collection-id injection untouched — unlike
`listField`, which injects per-item `id`/`rank` objects (`injectCollectionIds`
in `config_v2/server/internal/registry.ts` only processes fields that carry
`itemFields`). Use it when you need an ordered list of plain strings whose
exact contents and order matter (e.g. reorder directives `order` / `hidden`).

The settings-pane renderer is intentionally minimal: a textarea with one item
per line. It is the secondary view — the primary editor is wherever the strings
are produced (e.g. the in-app drag UI for reorder).

## Usage

```ts
import { stringListField } from "@plugins/config_v2/plugins/fields/plugins/string-list/core";

const myConfig = defineConfig("my-plugin", {
  fields: {
    order: stringListField({ label: "Order", default: [] }),
  },
});
```
