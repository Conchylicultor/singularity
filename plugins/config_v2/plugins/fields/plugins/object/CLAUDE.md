# object

Object field type for config_v2. Groups a fixed set of named sub-fields into
a single nested value. Unlike `listField` (variable-length array), this is a
fixed-structure object where every key is always present.

The web renderer wraps sub-fields in a collapsible section (default expanded)
and dispatches each sub-field via `FieldRenderer`. Nested `objectField`
instances work automatically through the recursive dispatch.

## Usage

```ts
import { objectField } from "@plugins/config_v2/plugins/fields/plugins/object/core";
import { textField, intField, boolField } from "@plugins/config_v2/plugins/fields/plugins/primitives/core";

const myConfig = defineConfig({
  fields: {
    connection: objectField({
      label: "Connection",
      description: "Server connection settings",
      subFields: {
        host: textField({ label: "Host", default: "localhost" }),
        port: intField({ label: "Port", default: 8080 }),
        tls: boolField({ label: "Enable TLS", default: false }),
      },
    }),
  },
});
```
