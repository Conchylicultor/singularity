# conv

Renders raw `conv-<id>` strings (e.g. `conv-1777406728-mb12`) inline as
clickable chips. Agents write the bare id in their text — no tag wrapping
needed — and the active-data linkify primitive replaces matches at render
time.

Clicking opens the referenced conversation in the right side pane alongside
the host (URL `/c/<host>/c/<side>`) via `convSidePane`. Falls back to
navigating full-screen to `/c/<id>` when the chip is rendered outside a
conversation context or self-references the host. The chip resolves the
title and current status via `useConversationById`; falls back to the raw id
if the conversation isn't in the index yet.

The id pattern matches conversation ids generated by
`plugins/conversations/server/internal/lifecycle.ts`
(`conv-<unix-seconds>-<4 base36 chars>`).
