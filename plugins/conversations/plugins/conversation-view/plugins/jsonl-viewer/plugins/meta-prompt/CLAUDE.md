# meta-prompt

Renders harness-injected prompt turns in the JSONL viewer. These are raw
`type:"user"` lines that Claude Code marks `isMeta:true` with no originating
tool call — loop/queue wakeup prompts, `"Continue from where you left off."`
resumes, and `<local-command-caveat>` blocks. They are never authored by the
human, so they render as a distinct dashed "Resumed by harness" block instead
of a normal user message.

The classification happens in the transcript parser
(`plugins/conversations/plugins/transcript-watcher/server/internal/parse-jsonl.ts`),
which emits `kind:"meta-prompt"` events that this plugin's `EventRenderer`
matches.
