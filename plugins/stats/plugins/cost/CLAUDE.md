# cost

Token usage and dollar cost across Claude Code sessions, sourced from
[`ccusage`](https://www.npmjs.com/package/ccusage). All data lives in
`~/.claude/projects/<project>/<sessionId>.jsonl`; ccusage parses the
files on demand. We never write to that directory.

The plugin joins ccusage's `SessionUsage.sessionId` with our
`_conversations.claudeSessionId` so charts can filter to
Singularity-launched sessions and the top-conversations table can show
real titles instead of bare UUIDs.

`singularityOnly` is a per-worktree config field (defaults to `true`).
Each chart reads it via `useConfigValues` and passes `?scope=...` to its
fetch URL so `useFetchJson` re-fetches when the toggle flips.
