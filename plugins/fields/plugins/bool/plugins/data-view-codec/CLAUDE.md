# bool / data-view-codec

The **bool** type's `data-view.value-codec` capability.

Contributes `DataViewSlots.ValueCodec({ match: "bool", codec })` — the codec a
DataView custom column uses to round-trip a native `boolean` cell value ↔ its
canonical text form in the generic `TEXT` storage column. Only `"true"` decodes
to `true`; a native boolean always encodes to `"true"`/`"false"` (so `false` is a
stored value, and the `BoolEditor`'s `!props.value` toggle stays correct — raw
`"false"` would otherwise be truthy). The read twin of the server-side bool
`::boolean` text cast.

  - Uses: `primitives/data-view.DataViewSlots`
