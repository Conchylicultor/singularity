# rank

The **rank** field type — a fractional-indexing string column backed by the
`rank_text` Postgres domain (`TEXT COLLATE "C"`) instead of plain `text`, so
ordering keys sort by byte order. Use it for any `defineEntity` table that needs
a sortable `rank` column (the field-record analogue of
`rankText("rank").notNull()`).

Registers ONLY its `FieldIdentity` (with `extends: textFieldType`) — it reuses
text's data-view cell / filter via the `extends`-chain fallback. The DB column
mapping (`rankText(name)`) lives in the `plugins/storage` sub-plugin; the
`rankField()` factory in `plugins/config`.

The value type is `string` (the raw key); the `Rank` value-object transform is a
wire-schema concern a consumer layers on with `@plugins/primitives/rank`'s
`RankSchema`, keeping `table.$inferSelect.rank` honest as `string`.
