# int / config

The **int** type's config capability: the `intField` factory (core) and the
integer number-stepper renderer contributed to `config-v2.fields.renderer` (web).

Relocated from `config_v2/plugins/fields/plugins/primitives`. The factory builds
a config_v2 `FieldDef<number>` (with `min`/`max`/`step`) carrying the canonical
`fields/` int token (`@plugins/fields/plugins/int/core`, id `"int"`); dispatch is
by id string. The renderer rounds on blur. Shared `FieldHeader` / `useLocalValue`
come from the slot owner (`@plugins/config_v2/plugins/fields/web`). `int` derives
its table/filter capabilities from `number` via the identity `extends` chain.
