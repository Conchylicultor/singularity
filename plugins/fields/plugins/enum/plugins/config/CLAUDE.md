# enum / config

The **enum** type's config capability: the `enumField` factory + `EnumFieldDef`
shape (core) and the `EnumRenderer` contributed to `config-v2.fields.renderer`
(web).

Relocated from `config_v2/plugins/fields/plugins/enum`. The factory builds a
config_v2 `FieldDef<string>` carrying the canonical `fields/` enum token
(`@plugins/fields/plugins/enum/core`, id `"enum"`) and preserves the manual
`meta` build with `typeHint: "Allowed values: …"`. The `EnumRenderer` renders a
radio group (≤3 options, or `display: "radio"`) or a dropdown (≥4 options, or
`display: "dropdown"`); its `.type` is the same token, so dispatch matches by id
string. `FieldHeader` comes from the slot owner
(`@plugins/config_v2/plugins/fields/web`).
