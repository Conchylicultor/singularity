# enum / config

The **enum** type's config-render capability — the `config-v2.fields.renderer`
contribution.

Relocated verbatim from `config_v2/plugins/fields/plugins/enum/web`: the
`EnumRenderer` renders a radio group (≤3 options, or `display: "radio"`) or a
dropdown (≥4 options, or `display: "dropdown"`). Its `.type` is the `fields/`
enum token (`@plugins/fields/plugins/enum/core`, id `"enum"`), and dispatch is by
id string, so it matches config descriptors built with config_v2's still-separate
`enumField` factory (also id `"enum"`). The descriptor shape (`options` /
`display`) is read via a type-only import of `EnumFieldDef` from config_v2's enum
core.
