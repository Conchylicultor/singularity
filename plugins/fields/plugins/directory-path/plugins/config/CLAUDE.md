# directory-path / config

The **directory-path** type's config capability: the `dirPathField` factory
(core) and the folder-picker renderer contributed to `config-v2.fields.renderer`
(web).

The renderer wraps `FolderPickerPopover` from the `folder-picker` primitive
(`@plugins/primitives/plugins/folder-picker/web`) — a typeable absolute-path
input with a live validity indicator and a "Browse" popover that walks the host
filesystem. The factory builds a config_v2 `FieldDef<string>` (schema
`z.string()`, default `""`) carrying the canonical `fields/` directory-path
token (id `"directory-path"`); dispatch is by id string. `FieldHeader` comes
from the slot owner (`@plugins/config_v2/plugins/fields/web`).
