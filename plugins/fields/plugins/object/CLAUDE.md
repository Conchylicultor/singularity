# object

The **object** field type of the unified fields primitive — groups a fixed set
of named sub-fields into a single nested value. Unlike `listField`
(variable-length array), this is a fixed-structure object where every key is
always present.

This barrel registers ONLY the `FieldIdentity` (label "Object", `MdDataObject`
icon) in the `fields.identity` registry. The value type is a plain
`Record<string, unknown>`. There is no `coerce` and no `extends`.

The config-render capability — the `objectField` factory plus the collapsible
sub-field renderer contributed to the frozen `config-v2.fields.renderer` slot —
lives in the `plugins/config` sub-plugin.
