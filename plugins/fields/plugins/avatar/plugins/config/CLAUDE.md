# avatar / config

The **avatar** type's config capability: the `avatarField` factory (core) and
the icon+color picker renderer contributed to `config-v2.fields.renderer` (web).

The factory builds a config_v2 `FieldDef<AvatarSpec>` carrying the canonical
`fields/` avatar token (`@plugins/fields/plugins/avatar/core`, id `"avatar"`);
dispatch is by id string. Its zod `avatarSpecSchema` runs the
`getFieldResolver("avatar")` transform so stored specs rehydrate their
`svgNodes`. The `AvatarSpec` / `SvgNode` value types come from the identity
core; `FieldHeader` comes from the slot owner
(`@plugins/config_v2/plugins/fields/web`).
