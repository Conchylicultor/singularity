# multiline-text / config

The **multiline-text** type's config capability: the `multilineTextField`
factory (core) and the `<textarea>` renderer contributed to
`config-v2.fields.renderer` (web).

Relocated from `config_v2/plugins/fields/plugins/multiline-text`. The factory
builds a config_v2 `FieldDef<string>` (with `rows`) carrying the canonical
`fields/` multiline-text token
(`@plugins/fields/plugins/multiline-text/core`, id `"multiline-text"`); dispatch
is by id string. The renderer defaults `rows` to `4`. Shared `FieldHeader` /
`useLocalValue` come from the slot owner
(`@plugins/config_v2/plugins/fields/web`). `multiline-text` derives its
table/filter capabilities from `text` via the identity `extends` chain.
