# number

The **number** field type — the base numeric type of the unified fields
primitive.

This barrel registers ONLY the `FieldIdentity` (label, icon, `coerce: Number`)
in the `fields.identity` registry. The type's data-view capabilities live in
dedicated sub-plugins under `plugins/`:

- `plugins/table` — the read-only `NumberCell` (`data-view.cell`).
- `plugins/filter` — the min/max `FilterContribution` (`data-view.filter`). The
  filter `Control` is carried for the future filter bar (task 2) but not rendered
  yet; its `predicate` / `isActive` are pure and applied in the data-view row
  pipeline.

Derived numeric types (e.g. `int`) declare `extends: numberFieldType` and reuse
these capabilities via the `extends`-chain fallback.
