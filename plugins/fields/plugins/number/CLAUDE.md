# number

The **number** field type — the base numeric type of the unified fields
primitive.

Registers its `FieldIdentity` (label, icon, `coerce: Number`) in the
`fields.identity` registry and contributes the two data-view capabilities for
the type: a read-only `NumberCell` (`data-view.cell`) and a min/max
`FilterContribution` (`data-view.filter`). The filter `Control` is carried for
the future filter bar (task 2) but not rendered yet; its `predicate` / `isActive`
are pure and applied in the data-view row pipeline.

Derived numeric types (e.g. `int`) declare `extends: numberFieldType` and reuse
these capabilities via the `extends`-chain fallback.
