# number / data-view-codec

The **number** type's `data-view.value-codec` capability.

Contributes `DataViewSlots.ValueCodec({ match: "number", codec })` — the codec a
DataView custom column uses to round-trip a native `number` cell value ↔ its
canonical text form in the generic `TEXT` storage column. Empty text decodes to
`null` (unset); `null`/`undefined` encodes to `""` (the empty-string-deletes
sentinel). The read twin of the server-side number `::numeric` text cast.

## Plugin reference

- Description: Number field type: data-view custom-column value codec (native number ↔ canonical text).
- Web:

  - Uses: `primitives/data-view.DataViewSlots`
