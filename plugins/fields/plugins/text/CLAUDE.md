# text

The **text** field type — the generic base text type of the unified fields
primitive.

This barrel registers ONLY the `FieldIdentity` (label, icon, string `coerce`)
in the `fields.identity` registry. The type's data-view capabilities live in
dedicated sub-plugins under `plugins/`:

- `plugins/table` — the read-only `TextCell` (`data-view.cell`).
- `plugins/filter` — the substring `FilterContribution` (`data-view.filter`).
- `plugins/inline` — the compact `TextEditor` (`data-view.cell-editor`).

Derived text types (e.g. `multiline-text`) declare `extends: textFieldType` and
reuse these capabilities via the `extends`-chain fallback.
