# color / config

The **color** type's config capability: the `colorField` factory (core) and the
popover color-picker renderer contributed to `config-v2.fields.renderer` (web).

The renderer wraps `ColorPickerPopover` from the `color-picker` primitive — a
popover with color area, hue slider, optional alpha slider, optional swatches,
and a format-cycling input (hex/oklch/hsl).

Relocated from `config_v2/plugins/fields/plugins/color`. The factory builds a
config_v2 `FieldDef<string>` (schema `z.string()`, default `"oklch(0 0 0)"`)
carrying the canonical `fields/` color token
(`@plugins/fields/plugins/color/core`, id `"color"`); dispatch is by id string.
`FieldHeader` comes from the slot owner
(`@plugins/config_v2/plugins/fields/web`).
