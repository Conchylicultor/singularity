# date / data-view-codec

The **date** type's `data-view.value-codec` capability.

Contributes `DataViewSlots.ValueCodec({ match: "date", codec })` — the codec a
DataView custom column uses to round-trip a native `Date` cell value ↔ its
canonical ISO text form in the generic `TEXT` storage column. Decoding yields a
real `Date` (so `coerce`, in-memory filter predicates, and sort all get a native
value — the plain `String(next)` path would corrupt it); an invalid/unparseable
date encodes to `""` rather than throwing. The read twin of the server-side date
`::timestamptz` text cast.

## Plugin reference

- Description: Date field type: data-view custom-column value codec (native Date ↔ canonical ISO text).
- Web:
  - Contributes: `DataViewSlots.ValueCodec` "date" → `dateCodec`
  - Uses: `primitives/data-view.DataViewSlots`
