# date

The **date** field type of the unified fields primitive — a data-view-only type
(no config-render capability).

This barrel registers ONLY the `FieldIdentity` (label "Date",
`MdCalendarToday` icon, `coerce` to an epoch millisecond timestamp) in the
`fields.identity` registry. The type's data-view capabilities live in dedicated
sub-plugins under `plugins/`:

- `plugins/table` — the read-only `DateCell` (`data-view.cell`), rendering the
  value as a relative "Nd ago" label.
- `plugins/filter` — the inclusive date-range `FilterContribution`
  (`data-view.filter`), with two native date inputs forming `[from, to]`.
