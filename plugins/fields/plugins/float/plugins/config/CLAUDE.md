# float / config

The **float** type's config capability: the `floatField` factory (core) and the
floating-point number renderer contributed to `config-v2.fields.renderer` (web).

Relocated from `config_v2/plugins/fields/plugins/primitives`. The factory builds
a config_v2 `FieldDef<number>` (with `min`/`max`/`step`) carrying the canonical
`fields/` float token (`@plugins/fields/plugins/float/core`, id `"float"`);
dispatch is by id string. The renderer commits the raw `Number(...)` on blur
(no rounding) and defaults `step` to `"any"`. Shared `FieldHeader` /
`useLocalValue` come from the slot owner
(`@plugins/config_v2/plugins/fields/web`). `float` derives its table/filter
capabilities from `number` via the identity `extends` chain.
