# header

The `header` reorder node type — the one **container** type shipped in this pass.
A labeled, collapsible box (`HeaderBox`) that renders its pre-rendered members as
`children`. It owns its payload schema (`label?`, `collapsed?`); the collapse
toggle writes back through `onPatch({ collapsed })`. The core tree format never
names `label`/`collapsed` — only this node type does.

Visual style mirrors the old group box (border + chevron header row) but without
the drag handle, rename input, delete button, or any endpoint/DB calls. Container
creation and membership are **config-only** this pass, so there is no `insert`.
Contributes one `ReorderNodes.NodeType(...)` to the `reorder.node-type` registry.
