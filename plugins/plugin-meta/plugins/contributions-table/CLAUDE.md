# contributions-table

Registry for the Studio **Contributions** aggregated-table surface. A pure
registry plugin (it contributes nothing itself): it owns the two `Contributions`
slots — `FacetTable` (one declarative aggregated table per facet) and `RowClick`
(a keyed row-click drill-down) — the `defineFacetTable` / `defineRowClick`
factories, and the `PluginChip`.

Living in the meta layer keeps its contributors meta: the
`facets/*/render-contributions` plugins contribute `FacetTable`, and app-side
pane owners (e.g. `studio/contributions/plugins/tables`) contribute `RowClick`.
A facet renderer never has to import an app pane. Consumed by the Studio
contributions view (`apps/studio/contributions`), which reads both slots and
wires clicks by matching a table's `facetId` to a `RowClick` handler.
