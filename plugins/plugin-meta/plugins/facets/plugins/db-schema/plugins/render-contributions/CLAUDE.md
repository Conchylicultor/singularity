# render-contributions

Contributes the interactive `Contributions.FacetTable` "Tables" tab to the Studio
catalog showing every `pgTable` aggregated across plugins, one row per table with
SQL name, TS var, and owning plugin columns. Reads `node.facets["db-schema"]` for
every plugin.

The table sets `onRowClick`, so clicking a row opens that table's live-SQL detail
pane (`catalog/plugins/tables/*` — columns, foreign keys, indexes, row count,
sample rows). Those detail sections are independent of facets and untouched; this
plugin only owns the aggregated list and the click-to-open wiring.
