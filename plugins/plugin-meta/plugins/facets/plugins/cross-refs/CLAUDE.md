# cross-refs

Extracts cross-plugin import relationships: per-runtime `apiUses` arrays
(server/central/web/core/shared) and the `importedBy` reverse index.

Uses `extract()` to scan each plugin's runtime directories for ES import
statements referencing other plugins' barrels. Uses `relate()` to invert the
apiUses graph into per-target `importedBy` lists — the first facet to
exercise the `relate()` lifecycle hook.

Dual-write: the monolithic extraction in `collectPlugin()` and
`computeRelationships()` still populates the legacy `PluginNode` fields.
The facet independently extracts the same data into `node.facets["cross-refs"]`.
The exported `crossRefsFacetDef` token lets other facets (e.g. `exports`)
read this facet's data via `getFacet(node, crossRefsFacetDef)` during their
own `relate()` pass.
