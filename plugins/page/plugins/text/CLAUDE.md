# text

Plain-text block type for the page editor. First block type to exercise the
full editor pipeline end-to-end.

Each text block renders an independent Lexical `PlainTextPlugin` composer.
Two internal Lexical plugins handle persistence and keyboard:

- **ValueSyncPlugin** — two-way sync between `useEditableField` (debounced
  persistence) and Lexical editor state. Uses the same `selfWriteRef` +
  `lastSerializedRef` + `queueMicrotask` pattern as the generic text-editor
  primitive.

- **KeyboardPlugin** — intercepts Enter (split), Backspace-at-start (merge),
  Tab/Shift-Tab (indent/outdent), and arrow keys at boundaries (focus
  navigation) and delegates to `BlockEditorAPI`.
