# heading-1

Heading 1 block type for the page editor. A thin renderer over the editor
plugin's reusable `BlockTextRenderer` primitive ā€” it stores `{ text }` and
declares three generic handle fields: `markdownPrefixes: ["# "]` (typing `# ` at
the start of a block converts it, prefix stripped and trailing text preserved),
`textVariant: "title"` (the shared editor renders the editable text at the
`title` typography variant), and `splitInto: "text"` (Enter at the END of the
heading produces a body `text` paragraph, Notion-style; mid-heading splits keep
the heading type).

Because it shares the same `BlockTextRenderer` as every other text-like block,
converting textā†”heading reconciles in place ā€” the live Lexical editor and its
caret survive. The conversions are driven generically by the editor's
markdown-shortcut and slash-menu plugins, which read every block type's metadata
