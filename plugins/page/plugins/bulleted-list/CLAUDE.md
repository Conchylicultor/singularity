# bulleted-list

Bulleted-list block type for the page editor. A thin renderer over the editor
plugin's reusable `BlockTextEditor` primitive — it supplies a `•` marker and
declares the CommonMark bullet markers (`* `, `- `, `+ `) as its
`markdownPrefixes`. Typing any of those at the start of a block converts it into
a bullet (the matched prefix is stripped, trailing text preserved); the
conversion is driven generically by the editor's markdown-shortcut plugin, which
reads every block type's declared prefixes from the dispatch slot.

Bullets nest with the editor's existing indent/outdent (Tab / Shift-Tab) — no
list-specific nesting logic.
