# toggle

Toggle (collapsible) block type for the page editor. A thin renderer over the
editor plugin's reusable `BlockTextRenderer` primitive — it stores `{ text }` and
declares two generic handle flags: `collapsible: "always"` (the shared editor
shows the collapse chevron even before the toggle has children) and
`splitChildWhenExpanded: { childType: "text" }` (when the toggle is expanded,
pressing Enter nests the split-off content as its first `text` child; when
collapsed, Enter splits into a sibling, like a bullet).

Typing `> ` at the start of a block converts it into a toggle (the matched prefix
is stripped, trailing text preserved), opening it by default; the conversion is
driven generically by the editor's markdown-shortcut plugin, which reads every
block type's declared `markdownPrefixes` from the dispatch slot.

Toggles nest with the editor's existing indent/outdent (Tab / Shift-Tab) — no
toggle-specific nesting logic.
