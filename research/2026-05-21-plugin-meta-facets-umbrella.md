# Facets Umbrella Plugin — Infrastructure Only

## Context

Step 2 of the facet-based docgen redesign (`research/2026-05-20-global-unified-facet-docgen.md`). Step 1 is done: `defineFacet()`, `getFacet()`, `setFacet()` exist in `plugin-tree/core`, and `PluginNode.facets` is initialized to `{}`.

This step creates the structural infrastructure — the umbrella plugin, the Facet interface, the discovery mechanism, and the pipeline hooks — without any actual facet implementations. Everything is wired but the facet registry is empty, so behavior is byte-identical.

## Design Decisions

- **Discovery**: `defineCollectedDir("facet")` — same pattern as check/lint/web
- **Extract context**: `{ node, pluginsRoot, modules? }` — dir-only, optional barrel modules
- **Facet types**: Live in `plugin-tree/core` (avoids circular dependency)
- **No re-exports**: `facets/core` does NOT re-export types from `plugin-tree/core` (boundary rule R10)

## File Changes

### 1. Extend `plugin-tree/core/internal/facets.ts`

Add `Facet`, `ExtractContext`, `RelateContext`, `DocContext` types. Import `Runtime`, `PluginNode`, `PluginTree` from `./plugin-tree` (same-package type import).

```typescript
import type { Runtime, PluginNode, PluginTree } from "./plugin-tree";

export interface ExtractContext {
  node: PluginNode;
  pluginsRoot: string;
  modules?: { mod: Record<string, unknown>; runtime: Runtime }[];
}

export interface RelateContext {
  tree: PluginTree;
}

export interface DocContext {
  node: PluginNode;
}

export interface Facet<T = unknown> {
  def: FacetDef<T>;
  extract(ctx: ExtractContext): T | undefined;
  relate?(ctx: RelateContext): void;
  renderDoc(data: T, ctx: DocContext): string[];
}
```

Export from `plugin-tree/core/index.ts`: `Facet`, `ExtractContext`, `RelateContext`, `DocContext`.

### 5. Wire docgen.ts — `codegen/core/docgen.ts`

```typescript
import { loadFacets } from "@plugins/plugin-meta/plugins/facets/core";

export async function buildEnrichedTree(root: string): Promise<PluginTree> {
  const facets = await loadFacets();
  const tree = buildPluginTree(resolve(root, "plugins"), { facets });
  await enrichPluginTreeDocs(tree, root, { facets });
  return tree;
}
```

## Execution Order

1. Types in `facets.ts` + exports in `plugin-tree/core/index.ts` (additive)
2. `buildPluginTree` + `enrichPluginTreeDocs` wiring (optional param, no breakage)
3. Known dirs updates (additive)
4. Create facets umbrella plugin (all new files)
5. Wire `docgen.ts` (import + pass facets)
6. `bun install` + `./singularity build`
