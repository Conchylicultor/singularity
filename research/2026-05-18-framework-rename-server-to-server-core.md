# Rename `plugins/framework/plugins/server/` ‚Üí `server-core/`

## Context

The framework plugin at `plugins/framework/plugins/server/` provides the server SDK ‚Äî `defineResource`, `defineServerContribution`, the Bun.serve entry point, and the plugin registry codegen output. Its name `server/` collides with the `server/` runtime directory convention that every plugin uses (e.g. `plugins/tasks/server/`). Renaming to `server-core/` disambiguates the framework primitive from the per-plugin runtime directories.

The `@server/*` tsconfig alias shields all ~163 consumer files from the rename ‚Äî no plugin source code changes are needed. The work is entirely in config, tooling, CLI, and docs.
